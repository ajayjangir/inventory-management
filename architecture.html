Request Lifecycle

  1. User changes a filter in FilterBar.vue → updates reactive refs in the useFilters composable.
  2. View component watch()es filter refs and calls api.js, which builds URLSearchParams from non-'all' values.
  3. Axios fires GET http://localhost:8001/api/...?warehouse=X&category=Y across CORS to FastAPI.
  4. FastAPI route handler in main.py reads query params, filters in-memory lists from mock_data.py via list comprehension.
  5. Pydantic response_model validates & serializes the filtered result → JSON response.
  6. Response lands in a Vue ref → computed properties re-derive chart data & aggregates → template re-renders.
