Request Data Flow

1

User Interaction

User changes a filter (Period, Warehouse, Category, Status) in FilterBar or navigates to a view.

2

useFilters Composable

Singleton reactive refs update. All views that call getCurrentFilters() pick up the new values.

3

api.js (Axios)

Builds URLSearchParams from filter object and issues a GET to localhost:8001/api/....

4

FastAPI Backend

Receives query params, runs apply_filters() and filter_by_month() on in-memory lists, validates via Pydantic.

5

Vue Renders

Response JSON updates reactive refs. Computed properties recalculate. Vue diffs and patches the DOM.
