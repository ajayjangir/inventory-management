Data Flow

1
User selects a filter in FilterBar.vue
Reactive refs in useFilters() composable update. All views watching filters re-execute their fetch.
2
View calls api.js function with current filters
getCurrentFilters() returns { warehouse, category, status, month }. api.js strips 'all' values before building query string.
3
FastAPI receives GET request with query params
Endpoint calls apply_filters() and/or filter_by_month() on the in-memory list loaded from JSON.
4
Filtered list serialized via Pydantic models
Response validated and typed. JSON returned to Axios client.
5
Vue reactive data updates → UI re-renders
Raw data stored in ref(). Computed properties derive aggregates, totals, and chart data from raw refs.
