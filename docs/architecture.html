Factory Inventory Management System

Architecture Overview — Vue 3 + FastAPI + JSON Data Store

Tech Stack

Frontend

Vue 3 Composition API Vue Router 4 Axios Vite
  • 7 view components (pages)
  • 9 reusable components
  • 3 composables (filters, auth, i18n)
  • i18n support (English / Japanese)
  • Dev server on port 3000

Backend

FastAPI Pydantic Python 3.14 Uvicorn
  • 15+ REST API endpoints
  • Pydantic model validation
  • In-memory data filtering
  • CORS enabled (dev mode)
  • API server on port 8001

Data Layer

JSON Files In-Memory Mock Data
  • 7 JSON data files
  • Loaded at server startup
  • No database (demo app)
  • Inventory, orders, spending, demand, backlog
  • Changes don't persist on restart
